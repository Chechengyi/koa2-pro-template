# koa2-mysql-sequelize-boilerplate-jwt

## 项目运行（nodejs 7.0+）
``` bash
# 克隆到本地
git clone https://github.com/Chechengyi/koa2-pro-template.git
# 进入文件夹
cd koa2-pro-template

# 安装依赖
npm i || yarn install

# 启动
npm run dev || yarn dev

# 发布环境
npm run prod || yarn prod
```

# 同步数据库表
```
npm run sync_database
```
